# 《残火长明》九灯、十二遗物与十二州对应表

本表用于统一地图、人物志、势力档案与剧情大纲。旧文档出现编号灯、旧州名或遗物归属冲突时，以本表和《全局设定定稿基准》为准。

## 一、九灯主节点

| 九灯 | 主节点 | 主要守护者 / 关联势力 | 核心职责 |
|---|---|---|---|
| 天灯 | 燧京州天京司灯台主塔 | 司灯台、司无咎 | 汇总八方灯讯，协调全阵 |
| 青灯 | 青州青灯塔 | 青灯一脉、灰炉民间守炉旧线 | 稳定东南地脉，监察长明残灯 |
| 寒灯 | 北寒州寒灯塔与北寒关封线 | 北境军、寒剑宗 | 阻断极北寒灾与归墟裂隙 |
| 南灯 | 南疆州白川祖林旧阵 | 南疆诸族、白川遗族 | 调和妖脉、灵河与南疆地脉 |
| 厄灯 | 渡厄州渡厄寺地宫 | 渡厄寺、慧尘 | 镇压魂毒、亡魂与记忆异变 |
| 云灯 | 云麓州云灯塔 | 宗门旧盟、司灯台档案官 | 保存盟约、校验各灯历史记录 |
| 楚灯 | 楚南州楚氏祖祠祭坛 | 楚氏世家 | 维持血脉阵术与南方封线 |
| 海灯 | 东海州潮眼海灯塔 | 东海商盟、镜海楼 | 监测潮眼、暗海与海上裂隙 |
| 陵灯 | 西陵州古战场陵灯塔 | 西陵遗民 | 镇守初裂旧战场并保存死者见证 |

灰炉镇地下埋有青灯副阵旧节点与长明残灯碎片，不是独立主灯。

归墟渊外镇渊阵本体是九灯共同指向的封印核心，不计入九灯名额。

## 二、十二州

| 行政州 | 常见旧称 / 简称 | 核心地点与职责 |
|---|---|---|
| 燧京州 | 中州、皇畿 | 天京、天灯、朝廷与司灯台中枢 |
| 青州 | 无 | 灰炉镇、青石城、青灯副线 |
| 北寒州 | 北境；旧称霜州、北庭州、寒川州 | 北寒关、三大防区、寒灯 |
| 南疆州 | 南疆十万山 | 白川祖林、南灯、诸族新盟 |
| 渡厄州 | 无灯之地 | 渡厄寺、厄灯、魂毒灾区 |
| 云麓州 | 云麓群山 | 云灯、宗门盟约与旧档 |
| 楚南州 | 楚州 | 楚氏祖地、楚灯 |
| 东海州 | 东海诸岛 | 潮眼、海灯、镜海楼 |
| 西陵州 | 西陵废土 | 归墟初裂古战场、陵灯 |
| 赤炎州 | 雍州旧地 | 赤炎炉城、雍州军府、军工补给 |
| 沧澜州 | 沧澜水乡 | 沉城、水脉污染与沧澜珠 |
| 黑沙州 | 黑沙荒漠 | 黑渊支脉、流放地与黑沙印 |

## 三、十二遗物

| 遗物 | 当前主线关联 | 功能与主题 |
|---|---|---|
| 长明残灯 | 林烬回应、灰炉镇旧节点 | 保存共同守誓残火，不属于皇室、林家或单一血脉 |
| 镇渊剑 | 寒剑宗、萧寒衣 | 斩断裂隙，也检验持剑者为何拔剑 |
| 渡厄莲 | 渡厄寺、沈青萝 | 分担与净化魂毒，不能绕过承受者同意 |
| 楚天璧 | 楚氏、楚明霜 | 稳定血脉阵术，也是祭器制度证据 |
| 白川骨笛 | 白璃、白川遗族 | 保存族群记忆与南灯旧誓 |
| 西陵碑心 | 西陵遗民、裴氏兄弟 | 记录初裂战争中被抹去的死者 |
| 观海镜 | 镜海楼、沈观潮 | 定位遗物、观测潮眼，不能提供完整未来 |
| 赤炎炉心 | 鲁重炉、赤炎匠师 | 锻造封印器具与战时民用设施 |
| 沧澜珠 | 沧澜水民 | 调节水脉并见证沉城污染 |
| 黑沙印 | 黑渊殿、黑沙州受害者 | 封锁归墟支脉，也是实验罪证 |
| 寒关铁令 | 北境军、顾长风 | 旧盟军誓与军民共同守关凭证 |
| 无名册 | 小满、医堂、驿卒与无名者 | 记录无名牺牲，是《残火录》的精神前身 |

十二遗物不是十二州的统治凭证，也不要求一州一物。终局后采用持有者、受影响者、见证者三方共管。
