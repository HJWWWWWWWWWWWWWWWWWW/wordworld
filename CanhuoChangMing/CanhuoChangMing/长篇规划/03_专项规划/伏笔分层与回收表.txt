# 《残火长明》伏笔分层与回收表

## 一、核心谜题释放节奏

| 结构区间 | 读者能够确认的真相 | 仍然保留的问题 |
|---|---|---|
| 第一部 | 灰炉镇灾变不是意外，林烬被标为适格者 | 谁下令、残火是什么 |
| 第二部 | 玄阳学院正在筛选遗物适格者 | 学院为谁服务 |
| 第三部 | 旧盟历史被删改，司灯台与宗门都参与隐瞒 | 九灯为何持续衰弱 |
| 第四部 | 妖族与南越同样参与过旧盟守封 | 谁抹去了他们 |
| 第五部 | 王朝以边境和军民承担守封代价 | 皇室是否掌控九灯 |
| 第六部 | 灯奴、祭器、送厄、实验属于同一套替代牺牲逻辑 | 是否存在不强制献祭的方案 |
| 第七部 | 残火不属于皇室，九灯可以异地回应 | 天下如何在没有中央命令时共同选择 |
| 第八部 | 九灯可以主动互认，林烬可以拒绝点火 | 替代方案能否持续 |
| 第九部 | 林烬在替代方案被真实尝试后选择点火 | 他的选择如何不被新权力利用 |

| 伏笔 | 首次埋设 | 中段推进 | 最终回收 |
|---|---|---|---|
| 灰炉镇铜梆 | 第一篇灾前报更异常 | 云灯档案显示警报被压下 | 终章废墟中无风轻晃，不确认由谁重新挂起 |
| 林家炉火 | 林照山教林烬守炉 | 证明灰炉民间守炉旧线曾客观维持青灯副阵 | 林烬以守炉方式理解最终点火，不获得血脉特权 |
| 适格者编号 | 黑松岭灯尸 | 学院试炼名单与灯籍对应 | 林烬公开拒绝适格等于义务 |
| 无名册 | 灰炉逃路残页 | 小满记录陆平、顾长风与各州死者 | 终局作为十二遗物之一记录启动与拒绝 |
| 镇魂灯 | 遗物试炼 | 显示旧盟献祭命令存在多版本 | 成为公开旧史的辅助证据，不代替十二遗物 |

| 寒关铁令 | 顾长风残缺军令 | 寒荒停战证明关外旧誓 | 维持终局北线撤离 |
| 楚天璧 | 楚明霜家族身份 | 历代祭器名册 | 切断楚灯血脉强制权限 |
| 云灯删档痕迹 | 雨夜行灾许可 | 各州命令来自同一中央权限 | 终局校验各地真实意愿 |
| 观海镜篡改 | 东海禁料航线 | 海听澜公开部分记录 | 定位九灯偏移 |
| 沈青萝魂毒 | 陆平救治 | 南疆疫潮、渡厄莲延缓 | 终局前救人后死亡，魂毒代价不消失 |
| 断风峡 | 北境撤离 | 萧寒衣旧案与镇渊剑 | 萧寒衣燃雪断后 |
| 废炉余温 | 灰炉镇日常 | 林烬多次想起守炉，各方提出公共借火处设想 | 终章只保留来源不明的暗红，不确认公共炉建成 |

## 三、伏笔管理规则

同一核心伏笔至少经过“出现、误读、验证、回收”四个阶段。

伏笔回收必须改变人物选择或局势，不能只提供背景知识。

若删除或移动篇章，必须同步检查该篇承担的伏笔是否失去中段推进。

终局不得突然新增能够解决九灯危机的遗物、血脉或规则。
