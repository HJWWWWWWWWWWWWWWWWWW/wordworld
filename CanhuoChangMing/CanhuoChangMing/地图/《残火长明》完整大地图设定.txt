# 《残火长明》世界地图与王朝格局设定

## 一、天下总格局

《残火长明》的世界名为：**玄尘大陆**。

玄尘大陆中央由一个大一统王朝统治，名为：

**大燧王朝。**

“大燧”之名取自“燧火不灭”。这个王朝并非普通的人间政权，而是当年第一次镇压归墟灾厄之后建立的天下秩序。

大燧王朝表面上统御州郡、镇守边关、册封宗门、掌控世家，实际上它真正的存在意义，是以整座天下为阵，维持一座覆盖玄尘大陆的古老封印。

这座封印被称为：

**九灯镇渊阵。**

王朝、州郡、边关、宗门、世家、禁地、古战场，全部都是这座大阵的一部分。

所以这个世界的核心矛盾不是简单的“正邪对立”，而是：

**天下能继续存在，是因为无数人被迫成为燃料。**

林烬身上的“残火”，正是这座大阵即将熄灭时，最后残留下来的火种。

---

## 二、大燧王朝总地图

```text
                          【极北冰原】
                              │
                    北寂汗庭 —— 寒荒白原
                              │
             【北境三关：北寒关 / 镇霜关 / 断风关】
                              │
        西荒古道 —— 赤炎州 —— 燧京州天京 —— 青州 —— 东海州诸岛
            │          │          │          │          │
        西陵州废土 —— 黑沙州 —— 楚南州 —— 渡厄荒城 —— 沧澜州
            │          │          │          │
        黑沙原 —— 南疆十万山 —— 南越诸部 —— 归墟暗海
                              │
                          【归墟渊】
```

大燧王朝统辖范围极广，名义上统御十二州；天京是燧京州内的皇都，不另算一州。外围则有北寂汗庭、南越诸部、远海岛群与西荒遗民等势力。

故事前期从青州灰炉镇开始，看似只是一个小地方的灾难。

但随着剧情推进，主角会逐渐发现：

灰炉镇被毁不是偶然。

北境战乱不是偶然。

渡厄州疫灾不是偶然。

楚家隐瞒真相不是偶然。

所有灾难的源头，都指向大燧王朝极力掩盖的归墟封印。

---

## 三、王朝核心：燧京州天京

中州位于玄尘大陆中央，是大燧王朝的心脏。

帝都名为：

**天京。**

天京是权力中心，也是九灯镇渊阵的中央阵眼。

### 核心地点

**天燧宫**

皇帝居所，也是镇渊大阵的中央阵眼。表面是皇权象征，实际镇压着归墟灾厄最核心的一道气脉。

**钦天监**

负责观测天象、归墟异动、灾厄征兆。钦天监掌握大量真相，但他们从不向天下公开。

**镇魔司**

王朝直属机构，负责处理邪教、妖变、灾厄、失控修士。镇魔司既是保护百姓的力量，也是王朝清除真相的刀。

**九灯台**

记录天下九处镇渊灯火的位置、状态与历代燃灯者姓名。但真正的名单被封存在禁卷中，普通人永远看不到。

**长明殿**

表面上是祭祀大燧先祖的地方，实际供奉着第一代残火宿主的残魂与遗物。

### 剧情定位

天京不是前期地图，而是中后期才开放的核心区域。

前期玩家看到的是废墟、灾民、边关、疫城。

中后期进入天京后，玩家才发现：

王朝高层早就知道灾厄会扩散。

他们也知道某些地方会被牺牲。

但他们选择维持“大局”。

大燧王朝不是纯粹反派，它确实守住了天下很多年。

但它的问题在于：

**它习惯了把活人当作阵法的一部分。**

林烬后期真正反抗的，不是“守护天下”这件事，而是反抗王朝把牺牲视为理所当然。

---

## 四、青州：主角起点

青州位于大燧王朝东部，是故事第一阶段的核心区域。

这里既靠近中州，又不像帝都那样被严密控制，所以最适合作为灾厄最早爆发的地方。

### 青州地图结构

```text
青州
│
├─ 青石城
│   ├─ 城南难民巷
│   ├─ 青石武场
│   ├─ 残火医堂
│   └─ 镇魔司青州分署
│
├─ 炉阳郡
│   ├─ 灰炉镇遗址
│   ├─ 旧炉地窖
│   └─ 槐安村
│
└─ 黑松岭
```

### 灰炉镇遗址

灰炉镇是林烬的故乡，也是残火苏醒之地。

多年以前，灰炉镇地下埋着青灯副阵旧节点与长明残灯碎片。后来归墟气泄露，副阵几乎熄灭，整座灰炉镇被灾厄吞没。

林烬在灾难中活下来，并不是因为幸运。

而是因为残火选择了他。

灰炉镇在剧情中具有双重意义：

前期，它是林烬失去一切的地方。

终局后，它是被救下的人重新点灯的地方。

### 槐安村

槐安村是灰炉镇灾民逃亡后的临时避难地。

这里适合承载小满的剧情，也适合展现普通人在灾难中的挣扎。

槐安村不是英雄诞生之地，而是让林烬第一次意识到：

他活下来之后，不可能再只为自己而活。

### 青石城

青石城是青州主城，也是前期主要活动据点。

这里有市井、武场、医馆、难民巷、官府、镇魔司分署。

青石城表面繁华，但城南难民巷堆满了从各地逃来的灾民。

它代表大燧王朝基层秩序的矛盾：

城中人仍在生活。

城外人已经开始死去。

### 残火医堂

残火医堂是沈青萝的核心地点。

这里不是单纯的治疗点，而是整部作品“救人”主题最早落地的地方。

沈青萝在这里救治灾民，也在这里第一次见到林烬体内残火的异常。

---

## 五、北寒州三大防区：战争与牺牲之地

北境不是单独一张雪地图，而是北寒州及关外寒荒的总称。

包括：

**霜岭防区、北庭防区、寒川防区。**

### 北境地图结构

```text
北寒州三大防区
│
├─ 霜岭防区
│   ├─ 寒剑宗
│   └─ 剑冢雪峰
│
├─ 北庭防区
│   ├─ 北寒关
│   ├─ 北境战场
│   └─ 断风峡
│
└─ 寒川防区
    ├─ 寒荒白原
    ├─ 寒川军民驿
    └─ 寒荒诸部旧营
```

### 北寒关

北寒关是王朝北线命门，也是顾长风的核心剧情地点。

这里常年抵御北寂汗庭南下，同时也承受极北归墟寒灾的侵蚀。

顾长风最终战死于北寒关。

他的死不是为了一个抽象的王朝，而是为了让关后的百姓多活一天。

### 寒剑宗

寒剑宗位于云麓州北境寒山，毗邻北寒州霜岭防区，是萧寒衣出身的宗门，也是宗门势力参与北境守封的前哨。

宗门讲规矩、讲剑心、讲清冷孤绝。

但北境战场教会萧寒衣：

剑不是为了证明自己无情，而是为了在必须有人留下时，替别人挡住风雪。

### 寒荒白原

寒荒白原是寒荒诸部、北境斥候与流亡边民共同活动的区域。

这里不是王朝真正重视的地方，寒荒诸部也不是被王朝平等对待的族群。

他们生活在边境，被归墟寒灾、北寂汗庭和王朝军令共同挤压。

北境军与寒荒诸部的剧情可以用来表现：

所谓天下大局里，边缘族群往往最先被牺牲。

### 断风峡

断风峡是萧寒衣最终断后的地方。

这里连接寒荒白原与北寒关，是难民撤退的最后通道。

萧寒衣死在这里，不是因为他突然变得慈悲，而是因为他很清楚：

他若退一步，身后所有人都会死。

---

## 六、赤炎州：西北军府与军工命脉

赤炎州位于王朝西北，是连接燧京州、西荒、北境的重要军事州。“雍州”是这片地区的旧称，沿用在雍州军府之名中。

### 地图结构

```text
赤炎州
│
├─ 雍州军府
├─ 铁衣营
├─ 西荒古道
├─ 孤烟堡
└─ 黄沙驿
```

### 剧情定位

赤炎州负责给北境输送粮草、兵器、援军。

如果赤炎州补给线断了，北寒关就守不住。

这里适合承载王朝军令与前线现实之间的冲突。

例如：

北寒关急需粮草，但燧京州命令雍州军府优先护送镇渊材料。

顾长风因此与军府产生矛盾。

这可以让他的战死更有重量。

他不是战死在单纯的敌人手里，而是死在“所有人都知道该救，但所有人都被规矩拖住”的局面里。

---

## 七、渡厄州：疫灾与弃民之地

渡厄城不应只是一个城市，而应该扩展为一整个被王朝半放弃的州。

正式名称：

**渡厄州。**

### 地图结构

```text
渡厄州
│
├─ 渡厄城
├─ 疫灾村落
├─ 无灯巷
├─ 封城门
├─ 残火医堂分堂
└─ 归墟教暗道
```

### 渡厄城

渡厄城原本是南方粮道重镇。

疫灾爆发后，王朝为了防止灾厄扩散，下令封城。

从那以后，渡厄城变成半死之地。

城里的人出不去。

城外的人不敢进来。

王朝史书会写：

“封渡厄，是为保天下。”

但城中百姓只知道：

他们被天下放弃了。

### 无灯巷

无灯巷是渡厄城最底层的弃民区。

这里没有官府救济，没有医馆药材，没有修士保护。

沈青萝进入渡厄州后，会在无灯巷救人。

她知道自己救不完所有人。

但她仍然不能停手。

因为她不是为了完成什么宏大的使命。

她只是无法看着眼前的人死去。

### 归墟教暗道

归墟教利用渡厄州作为疫毒实验场。

他们故意伪造南疆蛊毒痕迹，让王朝误以为疫灾来自南疆。

真相是：

疫毒来自归墟气。

渡厄州，是归墟教测试“人能被归墟侵蚀到什么程度”的地方。

---

## 八、楚南州：世家旧土

楚南州位于燧京州南面，是大燧王朝南方世家重地，民间常简称楚州。

这里是楚明霜人物线的核心区域。

### 地图结构

```text
楚南州
│
├─ 楚家祖宅
├─ 楚家祭坛
├─ 埋骨陵
├─ 断碑林
└─ 归墟裂隙
```

### 楚家祖宅

楚家不是普通世家。

楚家祖上参与过第一次镇压归墟，因此世代掌握封印秘密。

楚家表面臣服王朝，实际与皇室共同维护九灯镇渊阵。

### 楚家祭坛

楚家祭坛是九灯镇渊阵楚灯所在。

每一代楚家人都知道，有些秘密不能公开。

因为一旦公开，天下会动乱，王朝会失控，归墟教也会趁势而起。

但隐瞒的代价是：

很多无辜者被牺牲。

很多名字被抹去。

楚明霜的痛苦在于，她不能简单地否定楚家。

因为楚家确实守护过天下。

但她也不能原谅楚家。

因为楚家确实害死过很多人。

### 埋骨陵与断碑林

埋骨陵埋葬的是楚氏历代牺牲者。

断碑林记录的，则是那些不被允许留下名字的人。

楚明霜在这里完成真正的人物转折：

她不再只是反抗家族安排，而是决定亲手揭开家族的罪。

---

## 九、西陵废土：古战场与旧王朝遗骸

西陵废土位于王朝西部，是第一次归墟大战的古战场。

### 地图结构

```text
西陵废土
│
├─ 西陵古战场
├─ 断龙城
├─ 白骨驿
├─ 旧王陵
└─ 西渊裂谷
```

### 剧情定位

西陵废土是中后期真相线的重要地图。

王朝史书说：

大燧太祖在西陵击败邪魔，建立王朝。

但真相是：

当年真正封住归墟的，不只是太祖，而是一批被抹去姓名的残火宿主。

他们全部死在西陵。

他们的名字没有进入史书。

他们的坟也没有碑。

林烬在西陵废土第一次知道：

他不是第一个残火宿主。

在他之前，还有很多人被点燃过。

也有很多人被王朝当成工具用完之后遗忘。

---

## 十、南疆十万山：药火与古老真相

南疆位于王朝南方边缘，并不完全受大燧控制。

### 地图结构

```text
南疆十万山
│
├─ 白川祖林
│   ├─ 白川旧址
│   ├─ 断名石
│   └─ 白川救济所
├─ 万灵祭坛
├─ 南疆药谷
├─ 蛊王寨
├─ 万蛇沼
├─ 神木寨
└─ 南越诸部
```

### 剧情定位

南疆保存着比大燧王朝更古老的残火传说。

王朝认为残火是镇渊工具。

但南疆古祭司认为：

残火不是工具。

残火是人间不肯熄灭的意志。

沈青萝可以在南疆寻找治疗疫毒的药方。

但药方只能延缓归墟侵蚀，无法真正根除。

这会加深她后期的无力感。

南疆线的作用，是让主角团第一次听到与王朝完全不同的残火解释。

---

## 十一、东海诸岛：海贸、仙门与逃亡路线

东海是大燧王朝最自由、最复杂的地区。

### 地图结构

```text
东海诸岛
│
├─ 沧海港
├─ 浮舟岛
├─ 蓬莱旧观
├─ 海市
└─ 归墟暗海
```

### 沧海港

沧海港是王朝东部最大的海贸港口。

这里有商会、散修、海盗、海外仙门，也有从各州逃来的流亡者。

东海不像中州那样压抑，也不像北境那样苦寒。

它可以作为后期少有的喘息地图。

### 归墟暗海

归墟不仅在地下，也在海底。

东海深处有一片禁海，被称为归墟暗海。

这里连接最终的归墟渊。

后期主角团需要从东海寻找进入归墟暗域的方法。

这里也可以安排一次重要选择：

有人劝林烬离开大燧，去海外活下去。

但林烬最后没有走。

不是因为他圣母，而是因为他知道：

如果他走了，留下的人就没有路了。

---

## 十二、王朝之外的势力

### 北寂汗庭

北寂汗庭位于极北冰原以北，是游牧汗国。

它与大燧王朝常年交战。

但北寂汗庭不是单纯反派。

他们南下，是因为极北归墟寒灾扩散，草场、水源、部族营地都在死亡。

他们不是为了侵略而南下，而是被归墟逼着逃亡。

这会让北境战争更加悲凉。

顾长风杀死的敌人，也许只是从另一个方向逃命的人。

### 南越诸部

南越诸部位于南疆以南，名义上向大燧朝贡，实际上保持自治。

他们掌握古老药术和祭火仪式。

他们知道残火不是王朝独有之物。

南越古祭司可以告诉林烬：

残火从来不是皇室的东西。

它属于每一个在黑暗里还不肯跪下的人。

### 东海仙门

东海仙门曾经参与第一次镇渊，但后来退出王朝体系。

他们认为大燧王朝已经背离初衷。

他们愿意帮助林烬，却不愿意全面出兵。

因为他们也害怕归墟彻底复苏。

### 西荒遗民

西荒遗民是旧王朝后裔、流民、盗匪、残兵的混合势力。

王朝称他们为逆民。

但他们口中保留着被大燧删改过的历史。

他们知道大燧建国并不光彩。

他们可能才是最接近真相的人。

---

## 十三、九灯镇渊体系

九灯镇渊阵是整张大地图的核心。

大燧王朝不是随便建立在玄尘大陆中央的。

整座王朝，实际上是一座巨大阵法。

九盏镇渊灯分别位于：

```text
天灯：燧京州天京，司灯台主塔
青灯：青州青灯塔
寒灯：北寒州寒灯塔与北寒关封线
南灯：南疆州白川祖林旧阵
厄灯：渡厄州渡厄寺地宫
云灯：云麓州云灯塔
楚灯：楚南州楚氏祖祠祭坛
海灯：东海州潮眼海灯塔
陵灯：西陵州古战场陵灯塔
```

灰炉镇是青灯副阵旧节点；归墟渊外镇渊阵本体是九灯共同指向的封印核心，二者都不另算一盏主灯。

每一盏灯，都曾经有人以命点燃。

林烬身上的残火，就是九灯逐渐熄灭后残留下来的最后一缕火。

所以主线不是简单的升级旅行。

而是：

林烬一路走过九灯旧址，看见前人如何被点燃，如何被牺牲，如何被遗忘。

最终，他走到九灯共同指向的归墟封印核心前，选择以自己的方式燃尽残火。

王朝想让他作为工具死。

但林烬选择作为人承担可能无法归来的点火。

这就是他与王朝最大的区别。

## 十六、最终主题表达

大燧王朝看似统治天下，实际上是在用天下人的命续一盏快灭的灯。

林烬从青州废墟走出，一路经过边关、疫城、世家、古战场、帝都、东海，最后才明白：

所谓残火，不是王朝的工具。

它是那些不肯认命的人留下的最后一点光。

王朝想让林烬成为祭品。

归墟教想让世界归于沉寂。

而林烬最后选择承担可能失去一切的点火，不是因为他认命，也不是因为他甘愿被利用。

而是因为他想让那些被王朝放弃、被归墟吞没、被历史抹去的人，还能有明天。

所以《残火长明》的地图不是简单从小地方走到大地方。

它真正的路线是：

从废墟走向天下。

从天下走向深渊。

从深渊回到人间。

林烬失踪了，没人能够确认他是否还活着。

但他走过的地方，都还亮着。
