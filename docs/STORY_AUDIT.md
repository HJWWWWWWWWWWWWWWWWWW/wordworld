# 剧情关键节点审计

## 审计结论

主线现包含 38 个按章节实际顺序排列的关键阶段和 123 个可操作子节点，覆盖工作簿中的全部
15 个剧情篇章、21 个剧情里程碑，以及额外关键桥梁。

上一版虽然完成了里程碑名称覆盖，但存在阶段错位与过度合并。本轮已改为单一显式时间线，
不再通过多次列表插入拼接主线。

## 已修正的错位

- 退婚与立下三年战约发生在玄炉老人正式现身之前。
- 青莲源火争夺发生在炼药师大会之前。
- 炼药师大会发生在三年战约决战之前。
- 迦南学院、内院强榜、陨心源火和暗角域大战发生在重返沧澜、青岚宗大战之前。
- 进入中州、丹阁与三千星空火发生在营救玄炉老人之前。
- 中州立足与建立天元联盟拆分为不同阶段。
- 药典与远古种族联盟战发生在古帝洞府之前。
- 五帝破空从双帝之战中拆出，成为独立战后结局阶段。
- 第二次青岚宗大战修正为“击败青山、玄炉老人被掳”，为营救玄炉老人建立明确因果。
- 增加出云帝国战争、龙岛两阶段、黑渊决战、药族灭族战与营救林战。
- 菩提古树奖励补充“菩提心”，与工作簿成就条件一致。
- 陨心源火线修正为“炼气塔暴动与暗盟入侵”后，林烬被吞入塔底、炼化源火、破塔归来并清算冷煜。
- 暗角域拆为“首次穿越”“暗盟趁乱入侵”“炼化源火后清算冷煜”“重返暗角域”四段，避免冷煜线错位。
- 塔底古帝玉异动仅作为伏笔，不再提前确认古帝洞府。
- 中州开篇补回韩家、天北城、天目山血潭与四方阁大会，再接入风尊者和星落阁。
- 玄炉老人拆为“营救灵魂”“远古遗迹取灵圣骸骨”“复活玄炉老人”三段；星落阁在天墓归来后强化。
- 龙岛拆为“龙皇血脉”“三岛大战”“北龙王终战统一”三段。
- 黑渊殿拆为“血洗人殿削弱黑渊殿”和“净世白莲火后击败黑渊殿主”两段。
- 古帝洞府修正为玄族集齐八玉后开启，玩家不再获得、使用或从玄冥帝身上掉落古帝玉。
- 五帝破空补回数十年后开启源气通道、五帝并立和安排大陆后事的完整因果。
- 唯一剧情道具只在实际取得节点发放，重复效果不再产生虚假的“获得道具”日志。

## 防止再次错位

`STORY_CHAPTER_ANCHORS` 保存关键阶段的实际章节锚点。`PLOT_MILESTONE_SEQUENCE` 保存真实里程碑顺序，`PLOT_MILESTONE_COVERAGE` 将每个里程碑
映射到真实阶段和子节点。`CRITICAL_BRIDGE_COVERAGE` 额外保护玄炉老人被掳、毒宗战争、龙岛线、
黑渊决战、药族灭族战、营救林战与菩提心等因果桥梁。

自动化测试同时验证节点存在、完整主线可推进、里程碑顺序不会回退、剧情标记前置不会死锁，
引用的道具和人物关系确实存在，并使用稳定阶段 ID 迁移旧存档，避免新增阶段导致读档错位。

## 地图与早期修炼规则

地图开放由 `MAP_STORY_UNLOCKS` 统一控制。玩家即使等级足够，也必须推进到对应主线阶段后，
才能前往新区域。地图推荐等级只作为危险提示，不参与解锁。

玄炉老人苏醒前，修炼不会增加经验、灵力或境界等级，只会增加灵魂力量和冒险阅历。玄炉老人苏醒后，
修炼才恢复经验、灵力与灵魂的正常成长。
