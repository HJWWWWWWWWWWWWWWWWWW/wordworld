# 技能精通与功法效果设计

## 技能等级

- 技能最高 5 级，使用次数达到 10/30/60/100 次时提升至 2/3/4/5 级。
- 2、4 级强化基础倍率与暴击率；3、5 级获得流派强化或特殊效果。
- 所有现有技能及未来新增技能都会按元素与技能类型自动获得完整精通设计。

## 攻击技能里程碑

| 元素 | Lv3 强化 | Lv5 特殊效果 |
|---|---|---|
| 火 | 余烬：附加 2 回合灼烧 | 焚灭：伤害提高 25%，引爆灼烧 |
| 冰 | 寒侵：敌人减速 20% | 永冻：25% 概率封锁敌人下一次行动 |
| 雷 | 雷痕：无视 15% 防御 | 天罚：伤害提高 20%，20% 概率封锁行动 |
| 风 | 回风：返还 20% 灵力并增加连击 | 无影：伤害提高 15%，行动条推进 50% |
| 木 | 生息：恢复 5% 最大生命 | 共生：吸取实际伤害的 20% |
| 土 | 岩甲：获得 8% 最大生命护盾 | 山崩：无视 30% 防御并降低敌人防御 |
| 毒 | 蚀骨：额外中毒，中毒上限提升至 7 层 | 万毒噬心：每层中毒令伤害提高 8% |
| 无 | 凝练：降低消耗并提高伤害 | 返璞归真：提高暴击率与暴击伤害 |

## 辅助技能里程碑

- Lv3【运转自如】：灵力消耗降低 25%，治疗、护盾、回复、削弱等效果提高 25%。
- Lv5【圆满之境】：效果提高 50%，施放后行动条推进 50%。

## 功法技能

功法无法升级，装备后立即提供完整效果。除攻击、防御、生命、速度等基础属性外，
现已支持元素增伤、元素穿透、生命/灵力回复、开场护盾、领域减速/减防、闪避、
反伤、复活、额外连击、中毒强化以及冰冻、眩晕、束缚、震荡等控制效果。
