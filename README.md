# 残火长明 — 原创玄幻回合制 RPG

> **当前版本：v0.1-demo**  
> **状态：可试玩原型（开发中）**  
> **推荐启动：`python run_pygame.py`**

## 项目简介

《残火长明》是一款基于原创玄幻世界观构建的沉浸式回合制 RPG 游戏。

玩家扮演少年**林烬**，在灵玄大陆上修炼、探索、战斗、推进剧情，逐步成长。游戏采用传统 RPG 主循环：

1. 在据点休息、修炼和整理行囊
2. 前往已解锁区域探索，触发地图奇遇
3. 在回合制战斗中选择普攻、灵技、防御、丹药或逃跑
4. 获取历练、战利品和冒险阅历，推进剧情开放新地图
5. 消耗冒险阅历完成关键目标

## 运行环境

- **Python**: 3.9+
- **依赖**: 纯标准库（无第三方 pip 依赖）
- **可选**: pygame（用于图形版，`pip install pygame`）
- **平台**: Windows / macOS / Linux

## 安装与启动

```bash
# 克隆仓库
git clone <repo-url>
cd wordworld

# 安装 pygame（图形版需要）
pip install pygame

# 启动游戏
python run_pygame.py        # 图形版（pygame 瓦片地图 + 回合战斗）★ 推荐
python run.py               # 命令行版（纯文本终端）
python run_gui.py           # tkinter GUI 版（旧版）
python -m wordworld         # 等同于 python run.py
```

### Pygame 版操作说明

| 按键 | 功能 |
|------|------|
| 方向键 / WASD | 角色移动 |
| 空格 / 回车 | 交互（对话 / 开启宝箱 / 采集） |
| R | 休息（推进到次日清晨，恢复生命体力） |
| C | 修炼（消耗体力，运转灵力获得经验） |
| M | 菜单（角色状态 / 背包 / 装备 / 技能 / 存档读档） |
| F9 | 开启/关闭全部音效与氛围声 |
| F10 / F11 | 调低 / 调高音量 |
| ESC | 退出 |

### 命令行版操作说明

启动后根据终端提示输入数字选择行动。命令行版为调试和开发入口，推荐新玩家使用 Pygame 版获得完整体验。

## 新手流程（v0.1-demo）

首次进入游戏后，按以下步骤体验 20-30 分钟试玩：

1. **查看状态** — 打开菜单查看角色属性、背包、技能
2. **探索青石城** — 在出生地图中四处走动，触发事件和 NPC 对话
3. **完成族内试炼** — 达到等级 2，获得至少一次切磋胜利
4. **触发导师剧情** — 灵魂力量达到 6 后，戒指中的残魂现身
5. **修炼成长** — 消耗体力运转灵力获取经验，提升等级和境界
6. **解锁第二张地图** — 完成青石城主线后解锁「青石后山」
7. **探索后山** — 采集资源、遭遇野外战斗、击败小 Boss
8. **获得源火残片** — 首场 Boss 战后获得第一枚源火

> Demo 主线结束后会显示提示。你可以继续自由探索、修炼和战斗。

## 存档位置

存档文件保存在 `saves/` 目录下，JSON 格式。支持多存档槽位。启动时选择「继续游戏」即可读取最近存档。

## 项目结构

```text
wordworld/
├── run.py                  # 命令行版入口
├── run_pygame.py           # Pygame 图形版入口 ★ 推荐
├── run_gui.py              # tkinter GUI 版入口
├── src/wordworld/
│   ├── core/engine.py      # 游戏引擎（战斗/修炼/剧情/关系/技能/掉落）
│   ├── data/workbook.py    # Excel 配置解析与游戏数据加载
│   ├── data/*.py           # 各系统数据（装备/道具/技能/功法/源火/掉落/炼药）
│   ├── ui/console.py       # 命令行界面
│   ├── ui/pygame_ui.py     # Pygame 图形界面
│   ├── ui/gui.py           # tkinter 图形界面
│   ├── ui/tui.py           # 文本 UI（tkinter 终端风格）
│   └── config/paths.py     # 路径常量
├── story/                  # 剧情脚本与 Excel 配置文件
├── saves/                  # 玩家存档（JSON）
├── tests/                  # 单元测试（149 个）
├── scripts/                # 工具脚本（数据生成/迁移/补丁）
├── docs/                   # 设计文档与审计报告
└── CLAUDE.md               # AI 助手指南
```
