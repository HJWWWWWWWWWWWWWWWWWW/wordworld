# 残火长明 v0.1-demo 实施文档

## 第一卷「废柴与血夜」— 可试玩原型

> 对应剧情大纲：第一卷 第1章—第10章
> 核心情绪：少年受辱，家园破碎，但还没绝望——因为还有想保护的人

---

v0.1-demo 是项目从原型转为正式开发的第一站。

## 二、第一卷剧情阶段

### 阶段 1：废柴日常
- 林烬在青石镇醒来，残火微弱
- 查看角色状态，了解烬守体质
- 在镇内探索，与 NPC 互动（沈青萝、小满、林父）
- 了解世界基本设定：玄尘大陆、烬守遗族、残火

### 阶段 2：退婚
- 楚家来人，楚明霜退婚
- 林家在镇上颜面扫地
- 沈青萝仍然相信林烬
- 在大厅冲突中维护林家尊严

### 阶段 3：族内试炼
- 遗族年轻一代试炼开始
- 林烬以微弱残火勉强应战
- 击败至少一名对手证明自己
- 获得基础装备和技能

### 阶段 4：残火觉醒
- 试炼后残火濒临熄灭
- 戒指中萧寒衣残魂苏醒
- 萧寒衣揭示林烬体内沉睡的烬守天赋
- 学习残火诀（核心功法）
- 残火重燃，灵力开始流转

### 阶段 5：青石镇血夜
- 归墟魔气渗透青石镇
- 虚兽涌入镇内
- 林父为保护林烬和镇民战死
- 林烬第一次真正使用残火之力战斗
- 救下小满

### 阶段 6：离开故土
- 血夜过后，青石镇半毁
- 沈青萝、小满决定跟随林烬
- 萧寒衣指引前往玄阳学院
- 众人离开青石镇废墟
- **Demo 结束提示**：第一卷完，后续版本将开放玄阳学院与更广阔的世界

## 四、游戏内容范围

### 实现
- 青石镇地图（出生点、林家、演武场、市集、镇门）
- 3-4 种早期虚兽敌人
- 1 个小 Boss（血夜虚兽首领）
- 残火诀核心功法
- 3-5 个基础烬术
- 基础装备（林父遗物、沈青萝赠药）
- 青石镇 NPC 对话
- 剧情事件 x6

### 不包含
- 玄阳学院、其他城镇
- 遗物系统（23 遗物）
- 顾长风、白璃、叶无舟
- 归墟深渊

## 五、验收标准

1. python run_pygame.py 可启动
2. python -m wordworld 可启动
3. python -m compileall src 通过
4. python scripts/smoke_test.py 通过
5. 无原作专有名词残留
6. 窗口标题/UI 文案为残火长明
7. .gitignore 完整，临时文件已清理
8. CHANGELOG.md / TODO.md 存在
9. 新玩家 20-30 分钟试玩闭环
10. Demo 结束提示触发
